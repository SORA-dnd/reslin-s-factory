## What is Reslin's Factory?

Reslin's Factory is an AI-powered automation generator for Foundry VTT that helps DMs **save over 50% of their automation time**. It integrates with essential automation modules like **midi-qol**, **Chris's Premades (CPR)**, **DAE**, and **DFreds Convenient Effects** to create working automation code for your D&D 5e items, spells, and features.
